Special Instructions¶

It is probable that you will want to build an API endpoint in the mobile API that calls another API. This is very easy using the Python Requests module:

# ----------------------------------------
# Put this at the top of your python file
# ----------------------------------------
import requests

# ----------------------------------------------------------------
# Then put this in your function definition or a helper function
# ----------------------------------------------------------------
r = requests.get('https://[URL:PORT]/api/1.0/[ENDPOINT]')

The documentation for this is available here.
