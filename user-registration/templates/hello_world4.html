Now you are going to need to make the decision about which text editor you want to use. There are hundres to choose from but in Kenny's opinion, the best developers use Vim (or at least vim bindings). There is a learning curve but the basics are pretty easy to get started. Kenny loves to teach people Vim. Once you learn it you will appreciate how fast it is to do things with text. Also it's installed on almost every linux box so learning to use it is very helpful in your career.

Your development VM has two text editors installed that will work in the terminal. Vim and Nano.

You can also connect Notepad++ to the Virtual Machine via SSH if you prefer a GUI.

Mac users may want to use Sublime Text which also connects to servers via SSH via osxfuse and sshfs but it's a bit complex to setup.

If you love Visual Studio, you can download Visual Studio Code and follow These instructions to connect it to the virtual Server
