Now let's setup a couple of directories for our new hello world python flask app.

cd
mkdir rest_api
cd rest_api
mkdir myapp
ls

mkdir is a command to create (make) a new directory.
cd is the command to change directories.
ls is the command to list everything in a directory

The series of commands create two directories. That first `cd` puts you back in your home directory. I did that for the people that are coming from the Hello World tutorial. You will need two seperate directories for these two projects.
