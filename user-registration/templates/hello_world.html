It is important that you don't rush this tutorial. If you've never done this before it will take some time. That's ok. You are here to learn. The industry professionals are here to help you. If you've already built a couple Flask apps then you'll breeze through this tutorial. I still recommend you follow it because you'll write code that will be valuable on the project we are building.

You are going to need a terminal to run the Flask app. In Linux and Mac they are built in. In Windows we will need to download/save putty.exe. I typically creat a folder on the c:\ drive called `bin` and put it in there.

Windows - Putty 64bit or Putty 32bit (right click - save file as). The odds are that you want 64bit.

Linux & Mac - Open a Terminal.

Chromebook - Google `Chromebook Developer Mode` and follow those steps for a Chronos shell OR you can just install the SSH extension.
